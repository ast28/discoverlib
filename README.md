# Discoverlib

**"Planifica tus aventuras, a tu manera."**

Discoverlib es una aplicación Android moderna diseñada para ayudar a los usuarios a organizar, gestionar y optimizar sus viajes. Este proyecto permite a los usuarios planificar itinerarios diarios, hacer un seguimiento del presupuesto, gestionar actividades y almacenar recuerdos de sus viajes.

## Funcionalidades 

* **Tablero de inicio**: Consulta un resumen de tu próximo viaje y un calendario con la programación diaria.
* **Gestión de viajes**: Explora la lista de todos los viajes planificados, consulta itinerarios detallados y revisa el resumen del presupuesto (noches, precio total, actividades).
* **Descubrir actividades**: Explora y filtra actividades cercanas por categoría (Cultura, Comida, Naturaleza) para añadirlas a tu itinerario.
* **Detalles de la actividad**: Obtén información detallada sobre actividades específicas, incluyendo descripciones, duración, coste y mapas de ubicación.
* **Galería**: Visualiza y gestiona las fotos organizadas por viaje.
* **Preferencias**: Personaliza la experiencia de la aplicación con el modo oscuro, la selección de idioma y las preferencias de notificación.

## Tecnologías utilizadas

* **Plataforma**: Android (Min SDK: API 26)
* **Lenguaje**: Kotlin 2.0.21
* **Interfaz gráfica**: Jetpack Compose
* **Navegación**: Navigation Compose

## Estructura del repositorio

El proyecto sigue una arquitectura estructurada estándar:
* `/app/src/main/java/com/example/discoverlib/ui/`: Contiene todos los componentes de la interfaz de usuario (Pantallas, Scaffold, Tema).
* `/app/src/main/java/com/example/discoverlib/domain/`: Contiene los modelos de datos y la lógica de negocio.
* `/app/src/main/java/com/example/discoverlib/data/`: Contiene los repositorios de datos simulados (MockData).
* `/docs/`: Contiene la documentación del proyecto (modelos de datos, planificación del sprint, diseño, etc.).
