# Contribuir a Discoverlib

De momento se está utilizando una sola rama, `main`, para gestionar todo el código. Esto se debe a que desarrollo la aplicación de manera individual y no es necesario usar más ramas. En caso de colaborar más adelante con alguien, se crearían ramas adicionales (una para cada uno, o por funcionalidades) y se iría integrando en la rama `main` el trabajo distribuido.
